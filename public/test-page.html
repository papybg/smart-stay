Smart Stay API Тест

Login: POST /api/login

1. Проверка на базова връзка: GET /api/requests

2. Създай запитване: POST /api/inquiry

3. Избери свободна дата и одобри запитване: POST /api/requests/:id/approve

4. Маркирай като платено: POST /api/requests/:id/mark-paid

5. Откажи запитване: POST /api/requests/:id/cancel

6. Изтрий тестови данни: DELETE /api/requests/:id

Информация (много важно): Създаването на inquiry връща `request_code`. Въведи `request_id`, което получихте от API (id), за да тествате approve/paid/cancel.
